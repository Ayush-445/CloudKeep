☁️ CloudKeep

🔗 Live Demo: https://cloud-keep-22ws.vercel.app/

CloudKeep is a modern cloud storage web application that allows users to upload, manage, organize, and access their files securely from anywhere. It provides a clean UI, folder structure, file actions, and user profile management — similar to Google Drive but lightweight and developer-focused.

🚀 Features

📂 File Management Upload files (Images, Videos, PDFs) Create folders and organize files Nested folder navigation Grid & List view support

⭐ File Actions Star / Unstar important files Move files to Trash Restore deleted files Permanent delete option Download files anytime

🧭 Navigation & UI Folder breadcrumb navigation Tab system: All / Starred / Trash Drag & Drop upload support Responsive dashboard layout

👤 User Features Authentication using Clerk Profile update (name + avatar) Storage usage tracking (in MB)

🧠 Smart UI Components File previews (image/video/pdf icons) Empty state handling Loading states with spinner Confirmation modals for delete actions

🛠️ Tech Stack

Frontend Next.js (App Router) React Tailwind CSS HeroUI (UI Components) Lucide Icons

Backend Next.js API Routes Drizzle ORM NeonDB PostgreSQL

Authentication Clerk

File Storage ImageKit (for file uploads & CDN)
